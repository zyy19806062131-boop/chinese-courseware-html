# UNIT 1：认识你
**单元目标**：能和陌生人开始一段对话，交换基本信息，不卡壳、不沉默。

---

## 第 1 课：第一次见面

**真实场景**：在语言交换活动/课堂上第一次认识一个人，从打招呼到聊5分钟  
**今日微技能**：★ 附和词（backchanneling）——让对方知道你在听

### 语言零件（本课必学）

| 功能 | 表达 | 语用备注 |
|------|------|---------|
| 打招呼 | 你好！/ 嗨！ | "嗨"更年轻随意 |
| 问名字 | 你叫什么名字？/ 你叫什么？ | 后者更口语 |
| 说名字 | 我叫___。/ 我的名字是___。 | |
| 问哪里人 | 你是哪里人？/ 你从哪里来？ | |
| 问学中文多久 | 你学中文多久了？ | |
| 表示高兴认识 | 很高兴认识你！/ 认识你很高兴！ | |

### ★ 微技能教学：附和词

**为什么教**：中国人说话时，沉默=没在听=不礼貌。学生最大的问题是听懂了但不知道怎么反应。

**教学方法**：
1. 演示对比——老师说一句话，A组沉默，B组用附和词，学生感受差别
2. 明确给出清单：

| 场景 | 用法 | 例 |
|------|------|----|
| 表示在听 | 嗯 / 嗯嗯 | 对方说话时 |
| 表示同意 | 对对对 / 是是是 | 轻松随意 |
| 表示惊讶/兴趣 | 是吗？/ 真的？/ 真的假的？ | 语调上扬 |
| 问更多 | 然后呢？/ 后来呢？ | 让对方继续说 |
| 表示理解 | 哦，我知道了 / 哦，原来是这样 | |

3. 小练：老师说3件事，学生只用附和词回应，不说其他话

### 真聊任务

**角色A**：刚来的国际学生，第一天上课  
**角色B**：中国学生，来找语言交换伙伴  
**信息差**：A卡片有自己的信息（名字、国家、学习目标）；B卡片有B的信息，且B有问题要问A  
**老师设置的意外**：B说自己是广东人，A要问广东在哪里（测试A能不能自然接话）  
**产出**：录音，目标对话长度3分钟以上

### 自评重点
- 我用了几次附和词？（数一数）
- 对方说完一句话我有没有接话？

### 课后真实任务
用中文给一个中文母语者（真实的或网上找的）发一条自我介绍的语音，至少30秒。

## 第 2 课：交换联系方式

**真实场景**：认识新朋友后，想要保持联系，交换微信/电话/社交媒体  
**今日微技能**：★ 没听懂怎么救场（repair strategies）——比任何语法都重要

### 语言零件

| 功能 | 表达 | 语用备注 |
|------|------|---------|
| 提出加联系方式 | 我们加个微信吧？/ 可以给我你的微信吗？ | "吧"让提议更柔和 |
| 给号码/账号 | 我的微信是___。/ 我的电话是___。 | |
| 听不清号码 | 几几？/ 哪个？/ 再说一遍？ | |
| 数字表达 | 0-9 + 组合数字的说法 | 注意"幺"=1 in 电话号码 |
| 确认 | ___，对吗？/ 你说的是___？ | |
| 成功添加 | 我加你了！/ 好，加上了！ | |

### ★ 微技能教学：没听懂怎么救场

**为什么教**：学生听不懂就沉默是最大的口语障碍。有了救场工具，就没有"卡死"。

**工具清单**（按礼貌程度排序）：

| 情况 | 说法 | 注意 |
|------|------|------|
| 完全没听懂 | 不好意思，你能再说一遍吗？ | 最礼貌 |
| 没听清 | 什么？/ 你说什么？ | 随意，关系近时用 |
| 听懂了一部分 | ___是什么意思？ | 只问不懂的词 |
| 说太快了 | 能说慢一点吗？/ 能慢点说吗？ | |
| 想确认理解 | 你是说___，对吗？ | 复述+确认 |

**练习**：老师故意用模糊口音/说快/说方言词，学生必须用救场工具，不许沉默。

### 真聊任务

**角色A**：刚认识的新朋友，想加B的微信  
**角色B**：用一个复杂的微信号（含字母+数字，老师提前给B一张卡）  
**信息差**：A不知道B的微信号，需要问清楚  
**老师设置的意外**：B的电话号码有一个数字A会听错（老师用普通话里容易混淆的音：1/7，3/4）  
**成功标准**：A能正确写出B的微信号

### 自评重点
- 听不懂时，我用工具了还是沉默了？
- 我用了哪个救场工具？

### 课后真实任务
在一个语言交换App上（HelloTalk等）用中文给一个人发消息，尝试加对方为朋友。

## 第 3 课：聊共同话题

**真实场景**：认识了之后，怎么把对话继续下去，找到共同兴趣，聊10分钟不冷场  
**今日微技能**：★ 填充词/拖延（fillers）——给自己争取思考时间，避免冷场

### 语言零件

| 功能 | 表达 | 语用备注 |
|------|------|---------|
| 问兴趣爱好 | 你喜欢什么？/ 你平时喜欢做什么？ | |
| 说自己喜欢 | 我喜欢___。/ 我很喜欢___。 | |
| 表示共同兴趣 | 我也是！/ 我也喜欢！/ 真的？我也…！ | 关键：建立连接 |
| 问具体 | 你喜欢哪种？/ 比如呢？ | 深入话题 |
| 推荐 | 你应该试试___。/ 你听过___吗？ | |
| 转换话题 | 对了，你___呢？/ 说到这个，我想问… | |

### ★ 微技能教学：填充词/拖延

**为什么教**：学生说话时脑子在找词，填充词帮他们"占住话轮"，不被打断。

**中文填充词清单**：

| 填充词 | 使用场景 | 注意 |
|--------|---------|------|
| 那个… | 思考时 | 最常用 |
| 就是说… | 解释/换说法 | |
| 怎么说呢… | 找词/思考 | 很自然 |
| 嗯… / 啊… | 犹豫时 | 不要太多 |
| 你知道吗… | 引出话题 | |

**练习**：给学生一个话题，强制要求说30秒不停。遇到找不到词时必须用填充词，不许沉默超过3秒。

### 真聊任务

**角色A+B**：两个刚认识的人，在等人/等车，聊天打发时间  
**信息差**：每人有一张"我的兴趣卡"，对方不知道  
**老师设置的意外**：B说自己特别喜欢一个A没听说过的中国综艺节目名字，A需要继续聊而不是沉默（测试能不能用"你说的是什么？""那个节目讲什么？"继续对话）  
**成功标准**：对话持续5分钟，没有超过5秒的沉默

### 自评重点
- 我用了几次填充词？
- 我有没有主动找话题问问题，而不只是回答？

### 课后真实任务
找一个视频（抖音/B站），看一段中国人的日常对话，写下你听到的3个填充词。

## 第 4 课：★ Unit 1 真实任务日

**今天没有新内容——今天是"真实世界模拟"**

### 任务设计：语言交换活动

**场景**：模拟一个真实的"语言交换聚会（语言角）"，每人3-5分钟和不同人搭讪、聊天、交换联系方式  
**规则**：只说中文，不准说其他语言；不知道怎么说的词可以用肢体语言或画图  
**老师角色**：扮演一个"很热情但说话很快的中国阿姨"，主动找每个学生聊，测试他们的实战能力  

**评估维度**（老师现场打分，不是语法，是交际成功率）：

| 项目 | 0（没做到） | 1（做到了） |
|------|------------|------------|
| 主动开口打招呼 | | |
| 对方说话时有附和反应 | | |
| 听不懂时用了救场工具 | | |
| 成功交换了联系方式 | | |
| 对话超过3分钟 | | |
| 用了填充词（没有长时间沉默） | | |

**单元总结**：和学生一起回顾Unit 1的3个微技能，讨论"哪个最难用？为什么？"

# UNIT 2：吃喝玩乐
**单元目标**：能在日常生活场景（吃饭、买东西、出行）里独立完成任务，不依赖别人帮忙。

---

## 第 5 课：在餐厅点菜

**真实场景**：进入一家中国餐厅，从进门到付款全流程  
**今日微技能**：★ 应对意外（contingency）——真实生活从不按剧本走

### 语言零件

| 功能 | 表达 | 语用备注 |
|------|------|---------|
| 进门 | 你好！几位？/ 两位。/ 一个人。 | 服务员先问 |
| 要菜单 | 有菜单吗？/ 菜单给我看看。 | |
| 点菜 | 我要___。/ 给我来一个___。/ 这个怎么样？ | "来一个"最口语 |
| 问辣不辣 | 这个辣吗？/ 不要辣。/ 微辣可以。 | |
| 问价格 | 这个多少钱？ | |
| 结账 | 买单！/ 结账！/ 可以刷卡吗？ | "买单"最常用 |

### ★ 微技能教学：应对意外

**为什么教**：餐厅里意外每天发生。不会应对意外=每次出门都要找翻译。

**常见餐厅意外 + 应对话语**：

| 意外情况 | 服务员说 | 学生怎么接 |
|---------|---------|-----------|
| 菜卖完了 | 这个没有了 / 今天卖完了 | 那…有什么推荐的？/ 那我要___吧 |
| 没有位子 | 现在没位子，要等吗？ | 要等多久？/ 好，我等。/ 算了，谢谢 |
| 听错了 | 你要的是___吗？ | 不是，我要___。/ 对对，就是那个 |
| 要加钱 | 这个要加___元 | 好的 / 不加了，谢谢 |
| 不接受现金/只能扫码 | 我们不收现金 | 可以刷卡吗？/ 我用微信付 |

### 真聊任务

**角色A**：顾客（学生）  
**角色B**：服务员（另一学生或老师扮演）  
**信息差**：A有一张假菜单，但B有一张"今日没有/卖完"清单，里面有A可能点的菜  
**老师设置的意外**（B临场发挥选一个）：
- "这个菜今天没有了"
- "这个要等20分钟"
- "对不起，我们不接受现金，只能扫码"

**成功标准**：A最终成功点到菜并结账

### 自评重点
- 服务员说意外情况时，我有没有卡住超过5秒？
- 我用了什么方式解决意外？

### 课后真实任务
这周真的去一次中国餐厅（或点一次外卖打电话确认），全程用中文。

## 第 6 课：买东西/讨价还价

**真实场景**：在市场/小店买东西，从问价到成交（或放弃）  
**今日微技能**：★ 委婉软化（hedging）——不把话说死，更自然，不容易冒犯人

### 语言零件

| 功能 | 表达 | 语用备注 |
|------|------|---------|
| 问价 | 这个多少钱？/ 怎么卖？ | "怎么卖"含"可以讲价吗" |
| 嫌贵 | 有点贵。/ 能便宜一点吗？/ 打折吗？ | 语气要软，不要"太贵了！" |
| 还价 | ___块可以吗？/ 我给你___怎么样？ | |
| 表示犹豫 | 我想想。/ 等我看看。 | 给自己时间，也是策略 |
| 买了 | 好，就这个了。/ 我要了。 | |
| 不买 | 算了，谢谢。/ 下次吧。 | 不要干走，说一声 |

### ★ 微技能教学：委婉软化

**为什么教**：学生往往要么太直接（中国人觉得不礼貌），要么不敢说（觉得冒犯）。软化词解决这个问题。

**软化词工具箱**：

| 硬说法 | 软化后 | 效果 |
|--------|--------|------|
| 太贵了！ | 有点贵。/ 价格能不能低一点？ | 不冒犯，但意思一样 |
| 我不喜欢 | 我可能更喜欢___。/ 好像这个不太适合我 | |
| 你说错了 | 我觉得好像是___？ | 给对方台阶 |
| 我不会 | 我中文不太好，你能帮我吗？ | |
| 不要 | 现在不需要，谢谢。 | |

**关键词**：有点、好像、可能、大概、我觉得、也许

### 真聊任务

**角色A**：顾客，想买一件东西  
**角色B**：卖家（学生或老师），有底价  
**信息差**：A的卡片上有"我愿意出的最高价"；B的卡片上有"我的底价"和"今天的开价"  
**老师设置的意外**：B说"这个是最后一个了"（测试学生能不能用"那如果我今天买，能便宜吗？"等应对）  
**成功标准**：A和B达成一个价格（或A礼貌地放弃）

### 自评重点
- 我用了哪些软化词？
- 我讲价的时候语气是硬的还是自然的？

### 课后真实任务
看一段中国人讨价还价的视频（市场/直播带货），记录他们用了哪些软化词。

## 第 7 课：问路和交通

**真实场景**：在中国城市里问路、搭地铁、叫出租车/滴滴  
**今日微技能**：★ 确认理解（comprehension check）——确保听懂，不走错地方

### 语言零件

| 功能 | 表达 | 语用备注 |
|------|------|---------|
| 问路 | ___在哪里？/ 请问怎么去___？ | "请问"更礼貌 |
| 听懂方向词 | 左、右、直走、前面、后面、旁边、对面 | 这些必须熟练 |
| 叫出租/滴滴 | 我要去___。/ 走高速还是走普通路？ | |
| 地铁 | ___站怎么走？/ 在哪里换乘？ | |
| 确认目的地 | 是___路___号吗？/ 到了吗？ | |
| 感谢 | 谢谢！/ 麻烦你了！ | "麻烦你了"比"谢谢"更有人情味 |

### ★ 微技能教学：确认理解

**为什么教**：听别人说方向是最容易听错的场景——确认理解可以避免走错路。

**确认理解工具**：

| 方法 | 怎么说 |
|------|--------|
| 复述对方说的 | 你是说先走到___，然后左转，对吗？ |
| 总结关键信息 | 所以我要坐___号线，在___换乘，一共___站，对吗？ |
| 问最后一步 | 出了地铁怎么走？ |
| 确认时间 | 大概走几分钟？ |
| 不确认的后果 | （老师讲一个笑话：因为没确认走错路的故事） |

**练习**：老师念一段复杂路线（转3次），学生不能记笔记，只能用"确认理解"技巧当场确认。

### 真聊任务

**角色A**：迷路的外国人，手机没电，找不到目的地  
**角色B**：路人（或店员），知道怎么去  
**信息差**：B的卡片上有详细路线（含2-3个转弯），A要靠问和确认才能到达  
**老师设置的意外**：B给的方向里有一步"那个地方最近在修路，要绕道"  
**成功标准**：A能正确复述完整路线（老师验证）

### 自评重点
- 我有没有用"你是说…对吗？"确认过至少一次？
- 如果走错了，是哪一步没确认清楚？

### 课后真实任务
打开地图APP，用中文对着地图练习"给自己讲路线"，录一段30秒的自言自语导航。

## 第 8 课：★ Unit 2 真实任务日

### 任务设计：模拟"出门办一件事"

**场景**：学生抽签获得一个"今天的任务"，必须在模拟的"中国街道"里完成（教室布置成几个区）

**任务卡示例**：
- 任务A：去超市买3样东西，其中一样卖完了，要找替代品
- 任务B：迷路了，要找到一家餐厅并点到你想吃的菜
- 任务C：在市场买一件东西，把价格砍到开价的70%以下

**区域设置**：
- 餐厅区（1个学生扮演服务员）
- 市场区（1个学生扮演卖家）
- 街道区（老师扮演路人）

**规则**：每人有15分钟完成任务，全程录音，只说中文

# UNIT 3：社交与感情
**单元目标**：能处理更复杂的社交情境——邀请、拒绝、道歉、维持关系。

---

## 第 9 课：邀请朋友

**真实场景**：约朋友周末出去，对方可能有事/不想去，需要应对  
**今日微技能**：★ 礼貌拒绝（polite refusal）——"不"怎么说才不伤感情

### 语言零件

| 功能 | 表达 | 语用备注 |
|------|------|---------|
| 提出邀请 | 你周末有空吗？/ 我们一起去___怎么样？/ 你要不要来？ | |
| 接受 | 好啊！/ 没问题！/ 当然！/ 我很乐意！ | |
| 问细节 | 什么时候？在哪里？/ 还有谁去？ | |
| 拒绝（有事） | 那天我有事。/ 那天不行，我要___。 | |
| 拒绝（客气） | 下次吧。/ 我再看看。/ 不好意思，那天刚好… | |
| 提出新时间 | 那改天？/ 下周怎么样？ | |

### ★ 微技能教学：礼貌拒绝

**为什么教**：学生要么干脆说"不行"显得冷漠，要么不知道怎么拒绝就答应了不想去的事。

**中国人拒绝的方式（语用文化）**：

| 类型 | 例子 | 效果 |
|------|------|------|
| 给理由（有点模糊） | 那天可能有点事。 | 软性拒绝，留余地 |
| 表示遗憾 | 哎，真遗憾！/ 太可惜了！ | 让对方感觉你是真的想去 |
| 提出替代 | 这次不行，下次！ | 维持关系 |
| 模糊回应（文化点） | 嗯……我看看吧。/ 到时候再说。 | ≈ 不太想去，但不直接说不 |

**文化讨论**：中国人说"到时候再说"和"再看看"通常是什么意思？（答：多半是不去，但不想直说）

### 真聊任务

**角色A**：要邀请B去一件事（A卡片上写明是什么）  
**角色B**：不想去（B卡片上写原因），但要礼貌拒绝并保持关系  
**老师设置的意外**：A邀请两次，B要拒绝两次但态度越来越好（不能越来越冷淡）  
**成功标准**：拒绝完了，两人还是朋友（对话结束时还有笑声/约了下次）

### 自评重点
- 我的拒绝方式让对方感觉受伤了吗？
- 我有没有表示遗憾或提出替代？

## 第 10 课：道歉和解释

**真实场景**：迟到了、忘了一件事、无意中冒犯了人，要道歉并解释  
**今日微技能**：★ 自我修正（self-repair）——说错了怎么优雅地改回来

### 语言零件

| 功能 | 表达 | 语用备注 |
|------|------|---------|
| 轻度道歉 | 不好意思！/ 对不起！ | "不好意思"用于轻度；"对不起"更重 |
| 解释原因 | 因为___，所以___。/ 我以为___。 | |
| 承认错误 | 是我的错。/ 我不应该___。 | |
| 保证改正 | 下次我会___。/ 我以后注意。 | |
| 接受道歉 | 没事！/ 没关系！/ 算了。 | |
| 安慰 | 你不用太在意。/ 谁都有这种时候。 | |

### ★ 微技能教学：自我修正

**为什么教**：说话说到一半说错了，很多学生就停下来沉默，其实真实对话里自我修正是最正常的事。

**自我修正工具**：

| 情况 | 怎么说 |
|------|--------|
| 说错词 | 不是…我是说…/ 啊，我的意思是… |
| 说错意思 | 等一下，我重新说。/ 我换一种说法。 |
| 说到一半忘了 | 怎么说呢…/ 那个…（用填充词争取时间）|
| 意识到理解错了 | 哦！不是那个意思？那你是说…？ |

**练习**：老师故意给学生一个复杂情况描述，学生说到一半会说错，强制练习修正。

### 真聊任务

**角色A**：犯了一个错（迟到了1小时/忘记了约定），要向B道歉  
**角色B**：真的有点生气，但最终接受道歉  
**信息差**：A和B各有一张"当天发生了什么"的卡片，故事有出入（测试A能不能解释清楚）  
**老师设置的意外**：B说"你上次也是这样！"（A需要应对，不只是继续道歉）  
**成功标准**：B最终说"算了，没关系"

## 第 11 课：聊近况/闲聊

**真实场景**：跟认识的人再次见面，聊聊最近怎么样，维持关系  
**今日微技能**：★ 延伸话题（topic development）——怎么让对话继续下去

### 语言零件

| 功能 | 表达 | 语用备注 |
|------|------|---------|
| 问近况 | 最近怎么样？/ 最近忙吗？/ 这段时间去哪了？ | |
| 回答近况 | 还不错。/ 还好啊。/ 最近有点忙。/ 挺好的！ | "还好"="不太好"（文化点！）|
| 追问 | 忙什么呢？/ 为什么？/ 后来呢？ | 这些是"延伸话题"的核心 |
| 分享一件事 | 我最近___，你猜怎么着？/ 你不知道吧，我… | 制造悬念 |
| 表示共情 | 真的吗？/ 哇！/ 那怎么办？/ 听起来不容易！ | |

### ★ 微技能教学：延伸话题

**为什么教**：学生最大的问题是对话只有问答，不会"往深了聊"。

**延伸话题的两把钥匙**：

**钥匙1：问"追问问题"（follow-up questions）**
- 不要只说"哦，好的。"
- 要问：为什么？/ 然后呢？/ 你怎么想？/ 你喜欢吗？

**钥匙2：先附和再分享自己**
- 对方说完 → 先附和（真的？哇！） → 然后分享类似经历
- 例："我也有一次…"/ "我朋友也是这样！"

**练习**：给学生一个"无聊版对话"，让他们用两把钥匙改造它，变成一段"有来有往的真实对话"。

### 真聊任务

**角色A+B**：两个朋友，上周末都去了不同的地方，今天重新见面  
**信息差**：每人有一张"我周末发生的事"（包括一件有趣的和一件不太好的）  
**老师设置的意外**：其中一件事和上次对话相关（"上次你说你要去…，去了吗？"）  
**成功标准**：对话超过7分钟，两人都分享了至少一件事，都用了追问问题

## 第 13 课：打电话

**真实场景**：打电话预约（餐厅/医生/快递）、接陌生电话、打客服电话  
**今日微技能**：★ 话轮开启和结束（turn-taking: opening & closing）

### 语言零件

| 功能 | 表达 | 语用备注 |
|------|------|---------|
| 接电话 | 喂？/ 你好！/ 哪位？ | |
| 打电话开头 | 你好，我是___，我想问一下… | 先说身份再说事 |
| 转接 | 请问___在吗？/ 我找___。 | |
| 等待 | 好的，我等一下。/ 没关系，我等。 | |
| 预约 | 我想预约___。/ 几号有空？/ 下午两点可以吗？ | |
| 确认 | 好，那___点，我记下了。/ 谢谢，再见。 | |
| 结束通话 | 好的！那这样，谢谢！再见！ | 中国人结束电话的节奏很快 |

### ★ 微技能教学：话轮开启和结束

**为什么教**：打电话是口语里最难的场景——没有视觉信息，开头和结尾最容易出错。

**开启的结构**：
> 喂？→ 打招呼 → 说我是谁（如果对方不认识）→ 说我打来的目的

**结束的信号**：中国人怎么结束电话？
- "好，那就这样！"/ "好，好，好！"/ "行，那先这样！"
- 特点：会重复"好好好"，然后说再见，动作很快

**练习**：学生背对背（不看对方），只靠声音进行预约对话，感受"看不见对方"的挑战。

### 真聊任务

**角色A**：要打电话给一家餐厅预约周五晚上8点，共4人  
**角色B**：餐厅服务员（学生或老师）  
**老师设置的意外**：周五8点满了，要提出其他时间；或者要问清楚有没有忌口  
**成功标准**：A成功完成预约，并在挂电话前确认了时间、人数、姓名

### 课后真实任务
真的打一个中文电话（可以是订外卖、问快递、打一个不重要的客服电话），录音。

## 第 14 课：投诉和解决问题

**真实场景**：买的东西有问题要退换、服务不满意要投诉、房东有事情要解决  
**今日微技能**：★ 坚持立场的委婉说法（polite persistence）

### 语言零件

| 功能 | 表达 | 语用备注 |
|------|------|---------|
| 说明问题 | 这个有问题。/ 这个不对。/ 我买的不是这个。 | 直接说问题，不要绕 |
| 提出要求 | 我想退货。/ 能不能换一个？/ 能不能退款？ | |
| 对方拒绝后 | 但是…/ 可是…/ 我理解，不过… | 关键字：但是、不过 |
| 表示不满 | 这不太合理吧？/ 这样不太好吧？ | 语气软，意思清楚 |
| 要求升级 | 可以找你们经理吗？/ 我需要一个解决方案。 | |
| 达成解决 | 好，就这样，谢谢。/ 这样可以，谢谢你。 | |

### ★ 微技能教学：坚持立场的委婉说法

**为什么教**：学生要么太软（被拒绝一次就放弃），要么太硬（开始吵架）。

**"坚持但不冲突"的公式**：
> 承认对方的立场 + 但是/不过 + 重申自己的需求

| 对方说 | 硬回答（不好） | 软但坚持（好） |
|--------|--------------|--------------|
| 这是规定，不能退 | 我不管！我要退！ | 我理解，不过这个情况有点特殊… |
| 已经超过退货期了 | 那也要退！ | 嗯，我知道，但这个问题是刚发现的… |
| 我帮不了你 | 你为什么帮不了？ | 好，那能帮我找一个能帮我的人吗？ |

### 真聊任务

**角色A**：买了一双鞋，回家发现是两只右脚，要去换  
**角色B**：店员（老师），第一次拒绝，第二次给部分解决方案  
**信息差**：B的卡上有"店规"（不能退，只能换，但换的款式只剩大一号的）  
**成功标准**：A坚持了至少2次，最终达成某种解决方案（哪怕不完美）

## 第 15 课：讲一件事

**真实场景**：朋友问"你最近怎么样"，你想讲一件真实发生的事，让对方听得入迷  
**今日微技能**：★ 叙事节奏（narrative structure）——把一件事说得有人听

### 语言零件

| 功能 | 表达 | 语用备注 |
|------|------|---------|
| 开场制造悬念 | 你猜怎么着？/ 我跟你说一件事！/ 你不知道吧！ | 先抓注意力 |
| 交代背景 | 那是上个星期…/ 事情是这样的… | |
| 推进故事 | 然后…/ 后来…/ 结果…/ 突然… | 节奏词 |
| 高潮 | 你猜怎么了？/ 结果出乎意料… | |
| 结尾 | 最后…/ 就这样！/ 所以啊… | |
| 听者反应 | 然后呢？/ 后来怎么样了？/ 真的？！ | 听故事时用 |

### ★ 微技能教学：叙事节奏

**为什么教**：初级学生讲事情只会说"我昨天去了餐厅，然后吃了饭，然后回家了"——没有节奏感，对方不想听。

**好故事的结构**：
```
钩子（制造悬念）
 → 背景（在哪？什么时候？谁）
  → 发展（发生了什么）
   → 转折/高潮（意外！）
    → 结果/感想
```

**示范**：老师先用"没有节奏"版本讲同一件事，再用"有节奏"版本讲，学生说哪个更好听。

**练习**：给学生一张图（一个日常情景），要求用5个节奏词讲一个30秒的小故事。

### 真聊任务

**任务**：每人准备一件"最近真实发生的事"（可以是这学期学中文的经历），用叙事结构讲给对方听  
**角色B（听者）**：用之前学的附和词和追问问题回应，不许只点头  
**老师评估重点**：
- 有没有"钩子"开场
- 有没有至少一个转折
- 听者有没有被带入（有没有问"然后呢"）

### 任务设计

每个学生抽一张"任务卡"，独自完成，老师和助教在场观察（不帮忙）。
